## Ringkasan Konflik Kunci (prioritas tinggi)

1. `pending_events` dan `active_ripples` dimutasi oleh banyak modul (world/timers/npc/social handlers). Ini key queue kritikal.
2. `trace.trace_pct` adalah key dengan contention paling tinggi (npcs/timers/handlers/scenes/combat/hacking).
3. `economy.cash` dimutasi lintas domain (`economy.py`, `scenes.py`, `hacking.py`) dan dibaca mood/bio.
4. `world.locations.*` dimutasi lintas world/timers/security/scenes/hacking.
5. `inventory.r_hand/l_hand/active_weapon_id` dimutasi ganda oleh `inventory.py` + `inventory_ops.py`.

## Catatan Penggunaan Matrix untuk Refactor

- Saat memindahkan pipeline ke `engine/core/pipeline.py`, jaga urutan mutator yang menyentuh key konflik tinggi agar behavior tetap deterministik.
- Hindari menambah writer baru untuk key konflik tinggi tanpa test guard tambahan di `scripts/verify.py`.
